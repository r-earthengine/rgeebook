Earth Engine in Rstudio

This section requires the next libraries:

library(rgee)

ee_Initialize()

RStudio is the most popular R-based integrated development environment (IDE). It includes a console, syntax-highlighting editor that supports direct code execution, as well as tools for plotting, history, debugging, and workspace management. By combining Rstudio with rgee, users can quickly and effortlessly construct complex geospatial processes. The following features are available:

  • R code editor.
  • Map display for visualizing geospatial datasets.
  • API reference documentation (Using the ee_help addin).
  • Git-based Script Manager (Git tab).
  • Console output (Console tab).
  • Task Manager to handle long-running queries using the ee_manage_* functions.
  • Search of the data archive or saved scripts.
  • Geometry drawing tools, using rgee with **mapedit**.

GEE API reference documentation

The GEE API is quite extensive and users can easily get overwhelmed. To code more effectively, users need a tool that allows them to browse documentation without leaving the Rstudio IDE. Aware of this situation, the **ee_help** function was added to rgee to display the GEE documentation in an R-style format.

ee$Image()$geometry()$centroid %>% ee_help()
ee$Image()$geometry() %>% ee_help()
ee$Image %>% ee_help()
ee$Image %>% ee_help(browser = TRUE)


While ee_help is really helpful, a single keystroke, similar to the Rstudioâ€™s helper (just pressing F1!), would make the process even easier. This functionality is implemented in rgee through a Rstudio addin. To activate this feature, go to Tools -> Modify keyword shortcuts.



Then (1) type in the browser ee_help, (2) click the Shortcut column, and finally (3) press F3 or another desired keyword.


After terminating your Rstudio session you will now be able to display documentation with just a single keystroke (just pressing F3!).
